# 🎯 Jogo de Adivinhação (JavaScript)

Este é um **jogo de adivinhação de números**, onde o jogador precisa descobrir qual número o sistema gerou.

O projeto funciona diretamente no navegador e foi desenvolvido com **JavaScript para controlar toda a lógica do jogo**.

A estrutura inicial do projeto (**HTML e CSS**) foi fornecida pelo curso estudado.

# 🚀 Como funciona

O sistema gera **um número aleatório entre 1 e 100**, e o jogador precisa adivinhar qual é.

O jogador possui **até 10 tentativas** para acertar o número secreto.

A cada tentativa, o sistema informa se o número digitado é:

- 🔼 **Maior que o número secreto**
- 🔽 **Menor que o número secreto**

Se o jogador acertar, o jogo mostra uma mensagem de vitória.  
Se as **10 tentativas forem atingidas**, o jogador perde e o número secreto é revelado.

---

# 🎮 Regras do jogo

- O número secreto está entre **1 e 100**
- O jogador possui **10 tentativas**
- Após cada tentativa o sistema informa se o número é **maior ou menor**
- Quando o jogador **ganha ou perde**, o campo de entrada é bloqueado
- O jogador pode iniciar uma **nova partida clicando no botão "Novo Jogo"**

---

# 🧠 Funcionalidades implementadas em JavaScript

- Geração de número aleatório
- Controle de tentativas
- Armazenamento das jogadas do usuário
- Comparação entre número digitado e número secreto
- Exibição das mensagens de dica (maior ou menor)
- Bloqueio do campo de entrada ao finalizar o jogo
- Botão para reiniciar o jogo
- Atualização dinâmica das informações na interface
